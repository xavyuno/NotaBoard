# Task-Manager
A Milanote-like task manager

Keybinds:

	ctrl + z: Undo
	ctrl + c: Copy (Pastes item at mouse position)
	ctrl + d: Duplicate (Pastes item in same position)
	ctrl + r: Resets camera position
	delete: deletes selected item
	ctrl + LMB: select multiple items at once
	I: Zoom in
	O: Zoom out
ngl, idk if scrolling even works

Additional Information:

Items:
Link Item: When pasting the link of an image, it will try to display it
Board: Every board can have the same name but different ID, Each item placed inside of the board will be linked to the board's ID
Upload File: when selecting a file, it will try to display it's contents within the preview window, at a maximum of 100 lines. If the selected file is an image, it will try and display that image

Misc:
Holding RMB (Right mouse button) allows you to pan through the space, also works with middle mouse button and it captures your mouse in the process so moving isn't limited to your window (pretty lit if you ask me)
The notes you're reading is made from the same notes that you can drag and drop
Infact, the settings page uses the same Board system, all it does it change an already existing board within the program, Literally recycle, reuse, reduce or whatever

Information Collected/Stored:
only the items dragged and dropped, history and background color is stored into Save, removeHistory and settings files, respectively, basically anything you can physically see within the program
Files are not stored, only the directory leading to them and contents are turned into text and displayed within
